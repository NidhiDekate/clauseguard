# ClauseGuard

Understand a lease in under a minute — every finding backed by the exact clause it came from.

An AI decision-intelligence system built as a real multi-service application: a FastAPI backend running a LangGraph multi-agent pipeline (Planner → Retriever → Reviewer → Calculator → Report), a custom MCP server for clause retrieval, and a Streamlit frontend — the same architecture running here as in local development, packaged into one container for this deployment.
